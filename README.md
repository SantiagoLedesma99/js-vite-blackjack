 # BlackJack


 Paso para ejecutar el proyecto:

1. Iniciar Consola, Cd Clonar repositorio
2. Ejecutar el devServer "nmp run dev"

1. Ejecutar "" npm run build ""
2. tomar carpeta "dist" y desplegar.
